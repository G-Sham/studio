# **App Name**: KasparroAI

## Core Features:

- Home Page Value Proposition: Display a clear value proposition explaining Kasparro's AI-SEO platform in under 10 seconds.
- AI-SEO vs Traditional SEO Explanation: Explain the differences between AI-SEO and traditional SEO methodologies.
- Product Overview - Audit Pipeline Flow: Visualize the audit pipeline flow (Input -> Modules -> Output) on the platform overview page.
- Brand Dashboard Snapshot: Display key metrics such as AI Visibility Score, Trust / EEAT Score, and Non-Branded Keyword Coverage in a dashboard format.
- Audit Module Details: Show score, key insights, issues/flags, and recommendations for each of the 7 audit modules in the core product screen.
- System Architecture Visualization: Create a frontend representation (visual or structured text-based) of the InputAssembler, ContextPack, Audit modules, and Output surfaces.

## Style Guidelines:

- Primary color: A vibrant blue (#29ABE2) to represent innovation and trust.
- Background color: A light gray (#F5F5F5) to provide a clean and modern feel.
- Accent color: A bright green (#90EE90) for highlighting key insights and recommendations.
- Body and headline font: 'Inter' for a modern, machined and neutral look.
- Use simple, line-based icons to represent modules and data points.
- Employ a grid-based layout for the dashboard to ensure readability and a clear information hierarchy.
- Use subtle transitions and animations to enhance user experience when switching between modules and displaying data.
